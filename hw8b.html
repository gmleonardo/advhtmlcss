Comments about Tech Overview-Analytic Report

Google Analytics offers multiple reports; one of the most important concerning the user is the Tech overview report because it describes the essential characteristic of the device's users.
First of all, the period for this report is the last 90 days, and six panels confirm the information, and the first panel shows that users connect with a website (see figure 1) and do not associate with any apps.

Figure 2 is associated with the Operating System of the devices in which Android and iOS are predominant. This information is crucial because we can project an app focused mainly on mobile devices or more attention with the responsive design.

If we compare with figure 3, it is clear with a Pie graph is aligned with the info from figure 2, w here 63% are mobile, 34.8 % desktop, and only 2.2% tablet.

A critical problem when we define a website is the responsive design. Google Analytics offers a way to optimize that situation; the histogram (figure 4) shows the most predominant is 375x812 pixels, for example. It could be a better reference for @ media-max width.

Finally, what are the most critical Browsers? If we check figure 5, Chrome and Safari are predominant, so they can help us with the website performance when we build them.
